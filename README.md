# ResQNet 🚨

**ResQNet** is a premium, real-time emergency response and operations platform designed for hospitality and high-pressure environments. It transforms critical incident management from a manual process into a data-driven, hands-free experience.

![ResQNet Hero](https://images.unsplash.com/photo-1590402444582-43d16d35c717?auto=format&fit=crop&q=80&w=1200)

## ✨ Core Features

-   **🎙️ Hands-Free SOS**: Voice-activated emergency triggers (Keywords: "Fire", "Help", "Emergency") powered by the Web Speech API.
-   **📍 Live Radar Map**: Real-time Leaflet.js integration showing the precise location of every active incident.
-   **⚡ Live Monitoring**: A reactive command center feed that updates instantly as incidents are reported or assigned.
-   **📊 Operations Analytics**: Real-time calculation of "Average Response Time" and daily incident volume.
-   **🛡️ Responsive Dispatch**: Streamlined assignment flow with browser notification integration for responders.

## 🚀 Tech Stack

-   **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+)
-   **Backend**: Firebase Firestore (Real-time DB)
-   **Mapping**: Leaflet.js / OpenStreetMap
-   **UI Icons**: Lucide-Icons

## 🛠️ Setup & Local Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/ResQNet.git
    ```
2.  Open the directory:
    ```bash
    cd ResQNet
    ```
3.  Run a local server:
    ```bash
    # Using Python
    npm start
    # or
    python -m http.server 8000
    ```
4.  Open [http://localhost:8000](http://localhost:8000) in your browser.
